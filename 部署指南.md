# 贪吃蛇游戏 - 公网访问指南

## 方法一：使用本地服务器 + 内网穿透（推荐）

### 步骤1：启动本地服务器
1. 双击运行 `start_server.bat` 文件
2. 服务器将在 http://localhost:8000 启动
3. 保持这个窗口打开

### 步骤2：获取你的IP地址
打开命令提示符（CMD），运行以下命令：
```
ipconfig
```
找到你的IPv4地址，例如：192.168.1.100

### 步骤3：使用内网穿透工具（免费）

#### 选项A：使用 ngrok（推荐）
1. 下载 ngrok: https://ngrok.com/download
2. 注册账号并获取认证token
3. 运行命令：
   ```
   ngrok http 8000
   ```
4. ngrok会给你一个公网地址，例如：
   ```
   https://xxxx-xx-xx-xx-xx.ngrok-free.app
   ```
5. 将这个地址发给任何人，他们都可以访问你的游戏

#### 选项B：使用 cpolar（国内用户推荐）
1. 下载 cpolar: https://www.cpolar.com/
2. 注册账号
3. 运行命令：
   ```
   cpolar http 8000
   ```
4. cpolar会给你一个公网地址

#### 选项C：使用 frp（需要有自己的服务器）
如果你有云服务器，可以使用frp进行内网穿透

## 方法二：部署到免费云平台

### 选项A：GitHub Pages（完全免费）
1. 创建GitHub账号
2. 创建新仓库
3. 上传游戏文件（snake.html, snake.css, snake.js）
4. 在仓库设置中启用GitHub Pages
5. 获得永久免费的公网访问地址

### 选项B：Vercel（免费）
1. 访问 https://vercel.com/
2. 使用GitHub账号登录
3. 导入你的游戏仓库
4. 自动部署，获得公网访问地址

### 选项C：Netlify（免费）
1. 访问 https://www.netlify.com/
2. 拖拽游戏文件夹到网站
3. 自动部署，获得公网访问地址

## 方法三：使用Tomcat（需要安装）

如果你安装了Tomcat，可以：
1. 将游戏文件复制到 Tomcat/webapps/ROOT/ 目录
2. 启动Tomcat
3. 访问 http://your-ip:8080/snake.html

## 当前游戏文件
- snake.html - 游戏主页面
- snake.css - 样式文件
- snake.js - 游戏逻辑
- start_server.bat - 启动脚本

## 快速开始（最简单）
1. 运行 `start_server.bat`
2. 访问 http://localhost:8000/snake.html
3. 如果需要公网访问，下载并使用 ngrok 或 cpolar

## 注意事项
- 确保防火墙允许8000端口的入站连接
- 使用内网穿透工具时，请确保遵守相关服务条款
- GitHub Pages、Vercel、Netlify 提供的免费服务通常有流量限制


部署步骤
方法一: 通过Vercel CLI部署 (推荐)
安装Vercel CLI


bash
 
npm install -g vercel
登录Vercel


bash
 
vercel login
按照提示使用GitHub、GitLab或Bitbucket账号登录

部署项目


bash
 
vercel
第一次部署会让你输入项目名称
选择"Link to existing project"或"Create new project"
确认部署设置
完成部署

Vercel会自动构建和部署
部署完成后会给你一个公网URL,例如: https://snake-game-xxx.vercel.app
方法二: 通过Vercel网站部署
创建GitHub仓库


bash
 
git init
git add .
git commit -m "Initial commit"
git branch -M main
# 在GitHub上创建新仓库后执行:
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
访问Vercel

打开 https://vercel.com/
使用GitHub账号登录
导入项目

点击"Add New Project"
选择你的GitHub仓库
点击"Import"
配置部署

Project Name: 输入项目名称
Framework Preset: 选择"Other"
Root Directory: 保持默认 ./
点击"Deploy"
完成部署

等待几分钟,部署完成后会获得公网URL
